### 1) Remote control and async delegation are being split on purpose

Channels is different. It pushes messages from outside systems into the session that is already running. Anthropic’s docs call out two main uses:
- chat bridge from your phone via Telegram/Discord
- webhook receiver from CI, error trackers, deploy pipelines, and similar systems ([code.claude.com](https://code.claude.com/docs/en/channels))

- a **live control surface**

Handles ingress/egress.

- webhooks

### C) Intent Router

- is this **Live Control**
- **Dispatch**

- directives
- transcripts
- approvals
- blocked states
- summaries
- voicemail artifacts
- session snapshots
- queue items

This is what makes the system resumable.

### E) Executor Layer
Hands work to:
- OpenCode / Anti-Gravity
- background agents
- a local task queue
- selected repo/session

This is where your home machine actually does things.

### F) State + Audit Layer
Tracks:
- what’s running
- what completed
- what failed
- what awaits approval
- what was approved by voice
- what changed on disk
- what message was left for you
